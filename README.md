# Relay

A modern, open-source cross-platform device hub.

See project spec for details. This MVP implements Dashboard, Devices, secure pairing,
trusted-device reconnects, Clipboard sync, local file transfers, Device Details, and
Settings.

## Development

```bash
npm install
npm run dev           # vite dev server (renderer)
npm run electron:dev  # run full Electron app
npm run build         # typecheck + build
npm run lint          # lint
```

No telemetry, no ads, no forced accounts, local-first.
